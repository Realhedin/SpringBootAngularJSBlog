# SpringBootAngularJSBlog
Small blog application to try out SpringBoot technology

### Used technologies:
Spring Boot
<br>Java 8
<br>Spring 4
<br>Spring Data REST
<br>AngularJS
<br>InMemoryDB

### How to run
use command: <b>mvn clean spring-boot:run</b> from cmd
<br> or
<br> run ApplicationStart.java from IDE
